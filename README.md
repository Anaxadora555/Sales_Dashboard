## Sales_Dashboard
### 1. Sales_Dashboard - это интерактивный дашборд для отслеживания ключевых бизнес‑метрик компании, продающей офисное оборудование. Объединяет маркетинговые, финансовые и операционные показатели для всестороннего анализа продаж и эффективности бизнеса.

https://github.com/user-attachments/assets/376ec2ab-512f-4631-88f4-dc15d00b302a

### 2. __Цели:__
 * Обеспечить понятный и мгновенный доступ к ключевым метрикам для быстрого принятия решений.
 * Мониторить динамику продаж и прибыли, чтобы заметить тренды и колебания по месяцам.
 * Оценивать работу клиентских сегментов и регионов для более точного таргетинга маркетинга.
 * Следить за всеми расходами для улучшения рентабельности.
 * Находить и корректировать убыточные заказы для избежания больших потерь.
 * Формировать долгосрочные планы на основе аналитики по категориям товаров и регионам.

### 3. Визуализация:
   __KPI__ (Карточки) - 
      * Выручка
      * Прибыль
      * Активная База Клиентов
      * Средний чек
      * Рентабельность
      * Затраты на логистику
      
  __Динамика рентабельности__ (Столбчатая диаграмма) - 
     Рентабельность за выбранный период
     
  __Сегмент покупателей__ (Кольцевой график) -
     Категория клиентов
   
  __Рентабельность по регионам__ (Карта) -
     Экономическая эффективность в разрезе территорий
     
  __Прибыль и выручка по регионам__ (Таблица) - 
     Прибыльность по регионам
     
  __Номенклатура__ (Линейчатая диаграмма) - 
     Объёмы продаж по категориям товаров
     
  __Заказы с минусовой прибылью__ (Таблица) -
     Убыточные заказы по менеджерам

  ### 4. Этапы:
   * Встреча с представителями для понимания основных целей дашборда и определения ключевых метрик;
   * Сбор и подготовка данных: удаление лишней информации, соединение таблиц для дополнительных данных, создание вычисляемых полей;
   * Настройка мер для отображения карточек KPI;
   * Создание необходимых визуальных элементов:
   * Проверка и тестирование корректности перед запуском.
